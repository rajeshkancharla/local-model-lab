# local-model-lab
A structured benchmarking and evaluation framework for running Small Language Models locally using Ollama — covering inference performance, structured output validation, and multi-model comparison across edge deployment constraints.

local-model-lab is a hands-on engineering study of local SLM inference — benchmarking latency, enforcing structured outputs with Pydantic, and producing a data-driven model comparison report across Llama 3.2, Phi-4, and Mistral 7B. Built for real-world constraints: offline deployment, privacy requirements, and cost-conscious inference.
